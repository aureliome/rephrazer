# Rephrase Prompt Builder

Single-page React + TypeScript + Tailwind app that builds high-quality rephrasing prompts.

## Features
- Soft / Medium / Hard presets
- Checks grouped from softer → harder
- Tone & Formatting as checkboxes (“Adjust … if helpful”)
- Meaning control: “Adapt meaning if helpful” (otherwise preserve meaning)
- Output controls: bold-change highlighting, optional final comment
- Copy-to-clipboard prompt

## Quick start

```bash
npm install
npm run dev
```

## Tech
- Vite + React + TypeScript
- TailwindCSS
